# Load YMD/PMD Context

Load YMD/PMD format knowledge into your current session to enable authoring and validation.

## What This Does

This command loads the complete YMD/PMD format specification into your context, enabling you to:

- Create YMD orchestrator files
- Create PMD component files
- Validate composition structures
- Understand format rules and best practices

## Usage

Simply run this command to load the context. After loading, you'll be ready to work with YMD/PMD files.
